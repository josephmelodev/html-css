Usando links externos

Tag usada na aula

Tag a é usada para adicionar links a sua página ela possui dois atributos muito importantes, que são:

  • href que o indicado é colocar o valor _blank - fazendo abrir uma nova página com o click
  • rel com o valor external para o seo identifica que um link externo

Você pode usar meu repositório público no GitHub ou acessar minha página do youtube.

Exemplo do código:

<p>Você pode usar meu <a href="https://github.io" target="_blank">repositório público no GitHub</a> ou
acessar minha página do <a href="https://www.youtube.com" target="_blank" rel="external">youtube</a></p>
