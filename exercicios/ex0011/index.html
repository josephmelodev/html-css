Mídias de tamanhos diferentes

Testaremos 3 tamanhos de imagens diferentes com a tag <picture>, diminua e aumente a janela do seu navegador para testar

        
            <picture>
                <source media="(max-width: 750px)" srcset="imagens/p.png" type="image/png">
                <source media="(max-width: 1050px)" srcset="imagens/m.png" type="image/png">
                <img src="imagens/g.png" alt="imagem-grande">
            </picture>

na tag <source>, existe os atributos media="max-width: 000px" srcset="img.png" e type="image/png", sendo:

  • media = max-width que significa que a imagem selecionada ficara até o navegador atingir aquele tamanho maximo em pixels e min o tamanho minimo de pixels
  • srcset = você coloca o caminho do arquivo que será setado de acordo com o tamanho selecionado acima
  • type = verificar a lista de arquivos/tipo para não colocar errado
  • lembrar de seguir a ordem, maior para menor ou menor para maior
