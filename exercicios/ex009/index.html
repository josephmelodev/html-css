Trabalhando com listas

Todas as tags usadas na aula em lista

  • ul - un-ordened list - lista não ordenada (atributos: type="disc"/type="circle"/type="square")
  • ol - ordenet list - lista ordenada (atributos: type=a/A/1/I [definir numeração] e start="n" [o n é de onde começar a lista])
  • li - list item - item de lista - colocar para cara item da lista
  • dl - definition list - lista de definição
  • dt - definition term - termo de definição - termo que deverá ser definido na próxima tag
  • dd - description definition - descrição da definição - descrever o termo da tag de cima
